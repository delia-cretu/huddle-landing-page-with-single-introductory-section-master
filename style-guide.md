# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Violet: hsl(257, 40%, 49%) #674baf
- Soft Magenta: hsl(300, 69%, 71%) #e882e8
